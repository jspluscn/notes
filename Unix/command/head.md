> BSD General Commands Manual

# NAME
     head -- 显示文件前几行

# SYNOPSIS
     head [-n count | -c bytes] [file ...]

# DESCRIPTION
     显示前几行，默认是10

     其他参数和 tail 一样的

     选择多个文件，输出文件名 `==> XXX <==`

# EXIT STATUS
     The head utility exits 0 on success, and >0 if an error occurs.

# SEE ALSO
     tail(1)
