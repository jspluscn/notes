1.4. Accessing a Database

连接数据库：
- `psql`
- 图形化软件
- 编程语言

```
$ psql mydb
```

结果：
```
psql (10.1)
Type "help" for help.

mydb=#
```

简单的使用：
```
mydb=> SELECT version();
                                         version
------------------------------------------------------------------------------------------
 PostgreSQL 10.1 on x86_64-pc-linux-gnu, compiled by gcc (Debian 4.9.2-10) 4.9.2, 64-bit
(1 row)

mydb=> SELECT current_date;
    date
------------
 2016-01-07
(1 row)

mydb=> SELECT 2 + 2;
 ?column?
----------
        4
(1 row)
```
