# 8.2 [测试键值对是否存在及删除元素](https://github.com/Unknwon/the-way-to-go_ZH_CN/blob/master/eBook/08.2.md)

```go
// 测试 map1 中是否存在 key1
if val, ok = map1[key1]; ok {
	// do sth
}

// 从 map1 中删除 key1
// 如果 key1 不存在，该操作不会产生错误
delete(map1, key1)
```
